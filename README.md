# Heart-Healt
❤️ Heart Disease Dashboard - Power BI
📌 Project Overview

This Power BI dashboard analyzes heart disease patient data to identify survival rates, mortality trends, age-group patterns, and the impact of diabetes on patient outcomes. The dashboard provides interactive visualizations to support healthcare data analysis and decision-making.

🎯 Objectives
Analyze heart disease patient records.
Compare alive and deceased patient statistics.
Identify trends across different age groups.
Study the relationship between diabetes and survival rates.
Enable interactive filtering by gender.
📊 Dashboard Features
Alive Percentage KPI
Average Age KPI
Total Alive Count
Total Death Count
Age Group Analysis
Diabetes vs Total Alive Analysis
Gender Filter
Interactive Visualizations
🛠️ Tools & Technologies
Power BI
DAX
Data Modeling
Data Visualization
Excel/CSV Dataset
📈 Key Insights
Highest survival rate observed in specific age groups.
Diabetes shows a noticeable impact on patient outcomes.
Gender-based filtering helps identify demographic trends.
Interactive visuals make data exploration easier.
